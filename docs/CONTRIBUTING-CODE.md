# دليل المساهمة البرمجية (Contributing Code)

شكراً لاهتمامك بالمساهمة في تطوير مشروع **Im7o**! نرحب بكل المساهمات من المطورين.

## هيكل المشروع

المشروع مبني باستخدام **Next.js (App Router)** ويتكون من:
- `/src/app`: يحتوي على الصفحات الرئيسية (الرئيسية، صفحة صانع الألعاب، لوحة تحكم الأدمن، وموجه الألعاب).
- `/src/engines`: يحتوي على محركات الألعاب المختلفة (مثل `TurnBasedEngine`، `McqEngine`، `HotPotatoEngine`).
- `/src/components`: يحتوي على المكونات القابلة لإعادة الاستخدام (مثل `Timer`, `InstructionsModal`).
- `/src/data`: يحتوي على ملفات الألعاب (`games/`) والبطاقات (`cards/`).

## إضافة محرك ألعاب جديد (New Engine Template)

إذا كانت لديك فكرة لأسلوب لعب جديد كلياً، يمكنك برمجة محرك (Engine) جديد:

1. **إنشاء المحرك**: قم بإنشاء ملف جديد في `src/engines/` (مثلاً: `MyNewEngine.tsx`).
2. **تسجيل المحرك**: قم باستيراد المحرك وإضافته إلى الموجه في `src/engines/GameDispatcher.tsx`.
3. **دعم الواجهة**: أضف القالب الجديد إلى قائمة "نوع المحرك" في `src/app/creator/page.tsx`.
4. **دعم الإعدادات**: إذا كان المحرك يتطلب هيكلة بيانات خاصة للبطاقات (مثلاً البطاقات تحتوي على صور)، يجب تحديث شاشة "Creator Dashboard" لإتاحة إدخالها، وتحديث شاشة "Admin Dashboard" في `src/app/admin/page.tsx` لعرضها بوضوح.

## إرسال التعديلات (Pull Request)
1. قم بعمل Fork للمستودع.
2. أنشئ فرعاً جديداً (`git checkout -b feature/my-awesome-feature`).
3. قم بتطبيق التعديلات واحرص على توافقها مع نظام React / Next.js.
4. ارفع التعديلات وأنشئ Pull Request.
